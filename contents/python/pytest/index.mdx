[Pytest](https://docs.pytest.org)需要你已掌握 Python，如果还没有[请点击这里先去学习](../python)！

[Pytest](https://www.w3cschool.cn/pytest) 是一个成熟的全功能 Python 测试工具，可以帮助我们做自动化测试代码，以及生成测试报告等等。


## 安装
pytest属于三方包，需要额外安装
```bash
pip install pytest
```

## 创建你的第一个测试
新建一个 Python 脚本文件，输入以下代码
```py title="test_demo.py"
def test_add():
    assert 1 + 1 == 2
```

然后终端中运行 `pytest test_demo.py`，结果如下：包含断言测试统计、耗时等信息
```bash
PS pytest test_demo.py
==== test session starts === #测试开始
platform win32 -- Python 3.14.0... # 测试环境：win、python v3...
rootdir: D:\code\pytest-demo # 项目根目录
collected 1 item   #扫描到1个需要测试的case
test_demo.py .   [100%] # 测试完成（F 即表示运行失败 , . 符号即表示通过​）

==== 1 passed in 0.00s === #测试结束：耗时0秒，1个通过
```
