Python 是门强类型+动态类型的编程语言（不懂也没关系，后续再理解即可，不要死磕概念）。

主要用于编写后端服务、自动化任务（如爬虫、测试等）、大数据分析、C端和游戏开发！

Python 可称之为“老将新贵”——诞生于上世纪80年代末，却在近10年因 **AI/数据革命** 爆炸式流行。

- **生态雪球**：50w+ 包，一键 `pip install numpy` 秒变数据科学家。
- **社区企业**：Google、Meta、OpenAI 全押 Python。
- **对比语言**：C++ 快但难；Java 企业但冗长；JS 前端强但类型坑多。

一句话总结就是：语法简单且好用，而且大厂都在用！

目前均以 Python3 为主，不要再学习之前的老版本！
