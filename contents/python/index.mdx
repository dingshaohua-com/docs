**Python** 是一门计算机编程语言，在此之前或许你可能已经听说过很多种流行的编程语言，比如难学的 C，流行的 Java，适合网页编程的 JavaScript 等等。

那 Python 这门语言能干什么呢，它主要用于编写：后端服务、自动化任务（如爬虫、测试等）、大数据分析、客户端以及游戏等等！

其实 Python 可称之为 <span className='text-yellow-500'>老将新贵</span>， 虽诞生于上世纪 80 年代末，却在近 10 年因  **AI/数据革命**  爆炸式流，目前 Google、Meta、OpenAI 几乎全押 Python，不要担心学了会没用啦！

<Note>需要注意的是 目前主流均以 Python3 为主，不要再学习之前的老版本！</Note>
