因为机器只认识机器语言 比如`10111000 00000001`，而不认识 如python 等高级语言的代码，无法直接执代码，所以需要安装 python，python调用内部的解释器去执行python 代码即可。

先去[官网](https://www.python.org)，下载对应的安装包！

![](https://img.dingshaohua.com/doc-python/download-20251203204500.png)

## win

win下记得勾选上 注入环境变量，不然后续还得自己配，最后点击 Install Now 即可！

![](https://img.dingshaohua.com/doc-python/install-20251203205003.png)

打开终端输入 `py -V` ，返回类似如下，即代表安装成功！

```bash
Python 3.14.0
```

## mac

mac下安装，一直点击下一步即可

![](https://img.dingshaohua.com/doc-python/install-for-mac-20251203205128.png)

安装完成后，打开终端输入`python3 --version`，返回类似如下，即代表安装成功！

```bash
Python 3.14.0
```
