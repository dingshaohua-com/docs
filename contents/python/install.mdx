---
title: 安装
---

计算机硬件只能执行机器语言。Python作为高级语言，需要Python解释器来翻译和执行。因此我们需要安装Python环境（主要是解释器），才能在电脑上运行Python代码！

先去[官网](https://www.python.org/downloads)，下载对应的安装包！

![](https://img.dingshaohua.com/doc-python/download-20251203204500.png)

## win

win下记得勾选上 注入环境变量，不然后续还得自己配，最后点击 `Install Now` 即可！

![](https://img.dingshaohua.com/doc-python/install-20251203205003.png)

打开终端输入如下 ，返回类似`Python 3.14.0`，即代表安装成功！
```bash
py --version
```

## mac

mac下一直点击下一步即可安装，完成后打开终端输入如下，返回类似`Python 3.14.0`，即代表安装成功！

```bash
python3 --version
```
